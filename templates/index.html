Currency price service.

Go to {{ url_for("currency", date="10/09/2010") }} for daily prices.
Date format is dd/mm/yyyy.

Using tasks you can lookup history of prices during certain period.
Go to {{ url_for("tasks") }} for list of active tasks.

You can add (PUT), delete (DELETE), update (POST) and read (GET) tasks.
To add a task you want to send PUT request to tasks page. The rest of actions is performed on a specific task page (e.g. {{ url_for("task", id=1) }}
Task parameters consist of

  • date_from - first date in range
  • date_to - last date in range
  • cur_id - currency id: either USD or Euro
Parameters can be sent as json, xml or form (application/x-www-form-urlencoded). See examples of json and xml below.
Tasks and daily currency prices can also be received as json, xml, plain text or html depending on the Accept header of the request.

{
"date_from": "14/10/2010",
"date_to": "14/11/2010",
"cur_id": "Euro"
}

<task>
<date_from>10/11/2014</date_from>
<date_to>15/11/2014</date_to>
<cur_id>USD</cur_id>
</task>
